INSTALAÇÃO DE SISTEMA OPERACIONAI DE CÓDIGO ABERTO



Nesta aula, iremos aprender a como instalar a versão mais recente do Ubuntu (20.04 LTS), bem como o conceito de LTS e a manipulaçãp de sistemas de arquivos neste SO

Durante esta aula, alguns dos temas abordados também serão:

  • Pre requisitos para a instalação do sistema operacional para computadores com arquitetura 64 bits (minimo de RAM, capacidade de processamento, espaço em disco rígido)

  • Comandos básicos de manipulação do sistema de arquivo como: cat, cd, ls, cp, rm

Após a aula, você será capaz de instalar por conta própria sistemas operacionais Ubuntu em computadores, além de criar diretórios e arquivos, removê-los, copiá-los e verificar seus conteúdos
