INSTALAÇÃO DE SISTEMA OPERACIONAI DE CÓDIGO PROPRIETÁRIO



Nesta aula, iremos aprender a como instalar o Windows 10 sem chave de instalação, bem como a manipular seu sistemas de arquivos através do cmd

Durante esta aula, alguns dos temas abordados também serão:

  • Pre requisitos para a instalação do sistema operacional para computadores com arquitetura 64 e 32 bits (minimo de RAM, capacidade de processamento, espaço em disco rígido, resolução da tela)

  • Comandos básicos de manipulação do sistema de arquivo como: cls, cd, dir, mv, echo

Após a aula, você será capaz de instalar por conta própria sistemas operacionais Windows em computadores, além de criar diretórios e arquivos, removê-los, copiá-los e verificar seus conteúdos
