SISTEMAS OPERACIONAIS



É um conjunto de programas responsável por gerir os recursos do computador, como memória ram, memória de armazenamento, processamento, entre outras coisas. O sistema operacional reveza a sua execução com outros programas, para que desta forma ele possa tornar possível a execução dos programas, podendo possibilitar mais de um programa ser executado simultaneamente.

Segundo Tanenbaum os sistemas operacionais podem ser vistos de duas perspectivas

  • Em uma visão de cima para baixo (top down) Onde o hardware é uma abstração, fazendo o papel intermediário entre os programas e os componentes do hardware; ou

  • Em uma visão de baixo para cima (bottom-up) é um gerenciador de recursos, que as aplicações (Programas) a executar, como, quando e com quais recursos (memória, disco, periféricos).

De uma forma geral podemos ter a visão de que um sistema operacional serve para dar uma funcionalidade aos componentes de hardware
